Capper Timer - Distribution Package
====================================

QUICK START:
1. Double-click "CapperTimer.bat"
2. Press 'V' key to start the timer
3. Press 'V' again to cycle through timer durations (35s, 25s, 20s)

FEATURES:
- Overlay timer that appears on top of other windows
- Transparent background - only numbers visible
- Global hotkey: Press 'V' to start/cycle timers
- Network sync: All teammates see the same timer
- Visual warning: Timer turns red and flashes at 10 seconds
- Click-through: Timer doesn't block mouse clicks

TIMER DURATIONS:
- First press: 35 seconds
- Second press: 25 seconds  
- Third press: 20 seconds
- Fourth press: Back to 35 seconds (cycles)

TROUBLESHOOTING:
- If the timer doesn't appear, check if Windows Defender or antivirus blocked it
- Make sure you have internet connection for network sync
- If hotkey doesn't work, try running as Administrator
